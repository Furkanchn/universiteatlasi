# Vakıf Program Varyant Kontrolü

Kontrol tarihi: 2026-05-15

Bu rapor, yerel PostgreSQL verisindeki vakıf üniversitesi lisans programlarını burs/indirim/ücret varyantları açısından kontrol eder. Veri yazma, import veya scraping yapılmamıştır.

## Özet

- Vakıf program grubu sayısı: 2.121
- Sadece burslu görünen program grubu: 37
- Burslu + ücretli varyantı olan grup: 726
- Burslu + indirimli varyantı olan grup: 2.050
- Ücretli + indirimli varyantı olan grup: 698

## Not

“Sadece burslu” sonucu tek başına eksik veri kanıtı değildir. Bazı programlar YÖK Atlas’ta gerçekten yalnızca burslu açılmış olabilir. Bu liste, canlı YÖK Atlas ile hedefli doğrulanması gereken adayları gösterir.
